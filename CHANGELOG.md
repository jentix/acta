# Changelog

## 0.1.1

### MVP release

- Added the Acta CLI for initializing repositories, creating ADR/spec documents, listing, showing, validating, graphing, building artifacts and renumbering document IDs.
- Added `@acta-dev/core`, the canonical parser, schema, validation, graph, ordering, cache and artifact pipeline.
- Added the static Astro web viewer with documents, graph, validation, search, theme switching and English/Russian UI localization.
- Added Changesets release tooling and GitHub Pages deployment configuration for the dogfooding viewer.
